Here numpy is imported and a function is created to vectorize a sequence. First a numpy array that has rows=len(sequence) and column=dimension (10,000 by default) is initialized with 0s. Then for every element in the array we enumerate through them and set certain elements to 1 (while the rest remain 0).

In [5]:
import numpy as np
def vectorize_sequences(sequences, dimension=10000):
    results = np.zeros((len(sequences), dimension))
    for i, sequence in enumerate(sequences):
        results[i, sequence] = 1.
    return results

Overall, adjusting the hyperparameters of the model such as the number of hidden layers, the number of neurons in each layer, etc. can alter the performance of the model. Although the changes we made only made slight improvements (with some also making the model slightly worse), overall the model still had a tendency to perform much worse on the test data, indicating further changes that need to be made to prevent overfitting. One option is combining multiple of the thing we tried, such as adding more hidden layers and increasing the number of neurons. There is still a fine balance that needs to be found to get the best model.
